## AS 7 REGRAS DE OURO (resumo — leia em detalhe no gênero)

**REGRA 1 — O LIVRO ANFÍBIO (DENSIDADE PARA PODCAST)**
O livro precisa ser excelente para ler e excelente para ouvir. Não empilhe conceitos. Cada conceito técnico deve ter 3 batidas: explicação → analogia → tradução para o corpo. Nunca mais de 3-4 páginas de ciência sem respiro do Marcos.

**REGRA 2 — O LIVRO É UMA "BÍBLIA" (ZERO REFERÊNCIAS FANTASMAS)**
O livro deve ser 100% auto-suficiente. Proibido: "circula uma promessa", "a alegação do roteiro", "o protocolo secreto", "o spin de venda". O livro não responde a alegações que ele mesmo não apresentou.

**REGRA 3 — BISTURI, NÃO MARRETA**
Se você estiver corrigindo um capítulo, não reescreva do zero. Muita coisa já está boa. Atue como cirurgião: corrija o ponto exato, preserve o que funciona.

**REGRA 4 — ESTRUTURA DO PROTOCOLO**
O protocolo final deve ser quebrado em 2 grupos: "A base" e "Ferramentas situadas", com frase de transição.

**REGRA 5 — MEMÓRIA CURTA E REFERÊNCIAS**
O usuário é leigo e esquecido. Toda referência a capítulo anterior deve relembrar a ideia em meia frase.

**REGRA 6 — TERMOS TÉCNICOS EXPLICADOS PELA FUNÇÃO**
Todo termo técnico deve seguir: função em linguagem cotidiana → analogia visual → nome técnico → uso livre.

**REGRA 7 — CONCLUSÃO EM 3 CAMADAS**
Todo capítulo deve terminar com: (1) síntese do mecanismo, (2) extensão ao leitor, (3) abertura para o próximo capítulo.
