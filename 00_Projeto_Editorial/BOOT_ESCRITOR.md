Após ler os arquivos acima, sua tarefa é **aplicar as correções do relatório (item 10) nos 21 capítulos existentes**.

## REGRAS DE OURO (não esquecer)

- **Mantenha a voz.** O livro é narrativo, fluido, serve como audiolivro. As correções são de clareza e conclusão — não transformem o texto em manual.
- **Não reescreva do zero.** Os capítulos estão bons em qualidade geral. Aplique ajustes cirúrgicos onde o relatório indica.
- **Marcos é o caso composto recorrente.** Ele aparece dosado, sem virar protagonista. Mantenha assim.
- **Rick aparece só como epígrafe.** Nunca como personagem, nunca dialogando, nunca explicando conceitos.
- **Frases-âncora de uma linha com parcimônia.** A maioria das transições deve respirar em parágrafos desenvolvidos.
- **Fontes primárias nomeadas.** Todo estudo citado com autor+ano deve apontar para o periódico/DOI, não para jornalismo de divulgação ou sites comerciais.
- **Método de produção:** prefira o método monolítico (capítulo num fôlego só) ao modular. O modular gera problemas de costura.
