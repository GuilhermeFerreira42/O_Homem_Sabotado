# O Homem Sabotado — Projeto Editorial

## Status

Projeto iniciado. Esta pasta contém a produção do livro em arquivos Markdown separados por etapa e capítulo.

## Método de trabalho aprovado

- O usuário atua como **Editor-Chefe**.
- O assistente atua como **Orquestrador/Executor editorial**, aplicando a skill `BOOK-FORGE` salva no workspace.
- O livro será produzido em partes, por capítulos ou blocos de capítulo.
- Cada capítulo terá sua própria pasta.
- O conteúdo será escrito em Markdown para facilitar revisão, edição e montagem posterior.

## Direção narrativa

O livro deve manter a força dos roteiros do Rick Ensina, mas com organização de livro:

1. Abertura narrativa forte.
2. Exposição do problema.
3. Explicação biológica clara.
4. Contraste com a narrativa oficial/sistêmica.
5. Estudos, nomes, dados e fontes citadas.
6. Protocolo prático com cautela.
7. Separação entre fato documentado, hipótese plausível, extrapolação narrativa e orientação de segurança.

## Aviso editorial

Este manuscrito organiza e desenvolve um corpus de conteúdo sobre saúde, ambiente, hormônios e hábitos. Ele não substitui consulta médica, diagnóstico ou tratamento profissional. Quando houver medicamentos, sintomas persistentes, disfunção erétil, depressão, infertilidade, alterações hormonais relevantes ou suspeitas clínicas, o texto deve orientar o leitor a procurar um profissional qualificado.
