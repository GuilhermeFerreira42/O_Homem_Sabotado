# Template — Checklist de costura do capítulo

Use este arquivo depois de consolidar os módulos de um capítulo.

## Checklist obrigatório

- [ ] Há aforismos ou frases-âncora repetidos entre módulos?
- [ ] Há conceitos explicados duas vezes sem necessidade?
- [ ] Há listas ou enumerações repetidas?
- [ ] As emendas entre módulos têm contexto suficiente?
- [ ] Alguma transição parece corte seco?
- [ ] A linguagem está 100% no idioma do livro?
- [ ] Há termos em inglês que precisam ser traduzidos?
- [ ] Algum módulo está seco demais em relação ao ritmo do capítulo?
- [ ] O caso composto, se usado, aparece com função real e sem virar personagem artificial?
- [ ] O protocolo final nasce do mecanismo explicado ou parece colado?
- [ ] O fecho abre tensão para o próximo capítulo?
- [ ] As referências apontam para fontes primárias sempre que possível?
- [ ] O capítulo cobre todos os itens do mapa de cobertura?
