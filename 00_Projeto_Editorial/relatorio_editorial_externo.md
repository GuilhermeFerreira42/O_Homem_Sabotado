# Relatório Editorial Externo — Pesquisa de Forma e Identidade

Este arquivo registra o relatório editorial recebido do agente externo para orientar a forma definitiva do livro.

# RELATÓRIO EDITORIAL PROFUNDO
## Projeto: Livro de Saúde Masculina — Sabotagem Biológica do Homem Moderno

---

# RESUMO EXECUTIVO

Este relatório responde a uma pergunta central: **qual é a forma editorial que permite transformar um corpus de vídeos provocativos sobre saúde masculina em uma obra de não-ficção autônoma, séria, fluida e literariamente poderosa?**

A resposta curta, baseada em pesquisa editorial comparada, é:

> **Narrador investigativo direto em primeira pessoa + casos reais (ou compostos declarados) conduzindo cada capítulo + ciência introduzida pela experiência humana, nunca pelo conceito.**

O modelo mais próximo ao ideal para o projeto é o cruzamento entre *Nação Dopamina* (Anna Lembke), *Por que Zebras Não Têm Úlceras* (Robert Sapolsky) e *A Vida Imortal de Henrietta Lacks* (Rebecca Skloot) — com influência estrutural de *Outliers* (Gladwell) para o ritmo de revelação.

Rick não deve ser personagem fictício com diálogos completos. Peter não deve existir como personagem narrativo. Ambos devem ser transformados em estratégias de voz — e o relatório explica exatamente como.

---

# CONCLUSÃO OBJETIVA — QUAL CAMINHO SEGUIR

## Decisão 1: Modelo editorial
**Não-ficção narrativa investigativa**, com narrador em primeira pessoa estratégica, estrutura problema-revelação-protocolo, casos reais (ou compostos declarados) conduzindo cada capítulo.

## Decisão 2: Rick
**Epígrafe e voz provocadora ocasional** — não personagem. Sua função é ser a pergunta que o livro responde com rigor.

## Decisão 3: Peter
**Eliminado como personagem.** Suas funções são absorvidas pelo narrador, que assume as dúvidas do leitor em primeira pessoa.

## Decisão 4: Ciência
**Modelo de 6 passos:** situação humana → estranhamento → mecanismo (em imagem) → nomeação → implicção → evidência ao final. Hierarquia explícita: sabemos / evidências sugerem / hipótese plausível / especulação.

## Decisão 5: Estrutura de capítulo
**7 blocos:** Gancho → Ampliação → Revelação mecanística → Segunda camada → Limite → Protocolo → Fecho com tensão.

## Decisão 6: Voz
**Jornalista de ciência falando com amigo inteligente.** Primeira pessoa estratégica. Segunda pessoa para implicações práticas. Humor sardônico, não cômico. Indignação como conclusão, não como pressuposto.

## Decisão 7: Abertura recomendada
**Versão A ou Versão B**, dependendo da ênfase. Se o livro começa pela investigação epidemiológica → Versão A. Se começa pelo caso humano que humaniza o dado → Versão B. A Versão C é excelente para o prefácio ou para a nota do autor — não para a abertura do primeiro capítulo, pois o tom de manifesto pode assustar leitores que ainda não foram conquistados pelo argumento.
